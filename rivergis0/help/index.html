Welcome to RiverGIS!¶

RiverGIS is a QGIS plugin that helps you create HEC-RAS flow model and process its results. It is under active development and in pre-alpha stage. Currently only 2D models are supported.

RiverGIS is free/libre software and is released under the GNU General Public License. Browse the source code on github.

Requirements¶

For some operations RiverGIS needs an active connection to a PostGIS database. You can create a new connection from QGIS menu Layer ‣ Add layer ‣ Add PostGIS layers....

Loading HEC-RAS 2D results requires a h5py Python package ( http://www.h5py.org ). See Installation of required Python packages.

Import 1D river data from ISOKP system requires a MySQLdb Python package.

Some raster operations use SAGA and GRASS7 processing modules. Check if they are setup properly Processing ‣ Options ‣ Providers.
