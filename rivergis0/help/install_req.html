Installation of required Python packages¶

h5py¶

On Windows you can experience difficulties trying to install h5py for OSGeo4W Python. If you’re desperate, try the following:

  1. Install QGIS using OSGeo4W Network Installer — you need qgis from Desktop
  2. Download h5py from rivergis.com, unpack it and copy folder h5py to C:\Osgeo4W\apps\Python27\Lib\site-packages
  3. Restart QGIS and try to run Load WSEL from HDF from 2D results menu — if it asks you for a HDF file, the package is properly installed.

If the above doesn’t work for you, please, let me know.
