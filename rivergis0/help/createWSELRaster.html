Create Water Surface Raster¶

Menu: 2D Results

Creates water surface raster using previously loaded point WSEL layer (see Load WSEL from HDF). This tool uses SAGA’s raster module for interpolation

Create Water Surface Raster dialog window.

Tool parameters¶

  • Water surface elevation point layer — point layer with WSEL data.
  • Water surface elevation attribute — name of the attribute with WSEL
  • Target raster cell size — specify water surface raster cell size.
  • Tile size — SAGA raster module has a memory limitations and cannot deal with BIG rasters. If your raster is going to have more than 20 000 000 cells you need to divide the task and perform interpolation on smaller parts: RiverGIS does it on square tiles. Specify the tile size in pixels according to your data. The size shouldn’t be bigger than 3000.
  • Tile buffer size — there should be no gap between the tiles. Moreover, they have to overlap and that overlapping distance is the buffer size.
